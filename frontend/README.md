# Akademi Eğitim Takip Sistemi - Frontend

React tabanlı frontend uygulaması.

## Kurulum

```bash
cd frontend
npm install
```

## Çalıştırma

```bash
npm start
```

Uygulama `http://localhost:3000` adresinde çalışacaktır.

## Özellikler

- Eğitim listeleme (sayfalama ve filtreleme)
- Eğitim ekleme/düzenleme formu
- Multi-select ile kategori, eğitmen, sorumlu ve paydaş seçimi
- Eğitim detay ekranı (tüm ilişkileri gösteren panel)
- Proje bağlantısı görüntüleme
- JWT authentication

## Backend Bağlantısı

Backend `http://localhost:8080` adresinde çalışmalıdır.
